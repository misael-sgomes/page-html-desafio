Para que serve a âncora?

Home

A tag âncora, ou , é um elemento do HTML que permite criar hiperlinks, ou seja, links clicáveis, que podem ser usados para apontar para outros domínios ou para páginas do próprio site

Uma âncora pode apontar para qualquer recurso na Web: uma página em HTML, uma imagem, um arquivo de som, um filme, etc.

Como usar a âncora para linkar?

Home

A tag ‹a› é usada para criar uma âncora de onde vincular, depois é necessário um atributo chamado href que é usado para endereçar o documento, ou seja vincular e dessa forma as palavras entre as tags de abertura e fechamento da âncora serão exibidas como um hipervínculo.

Exemplo de uma âncora: ‹a href="#nome-do-id"›Titulo que ficará com hiperlinks‹/a›

Importante ressaltar que para o funcionamento das ancoragens é necessário que tenha um ID com o mesmo nome do href descrito no conteúdo a qual será feito vinculo. Dessa forma o HTML entende para onde direcionará o usuário ao clicar nesse link.

Observações Complementares

Home

A âncora é útil para facilitar a navegação em páginas com muito conteúdo, pois leva o usuário diretamente para a informação que ele pesquisou.

Para ancorar a uma outra página do seu site ou externo, podemos usar um link https://...

Para ancorar dentro da própria página, sempre lembrar de usar no href o # pois dessa forma você indica que ele vai ler um id dentro da sua página e assim direcionar para o mesmo.
